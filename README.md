# LinkHub - Custom Link-in-Bio Tool

A customizable and monetizable link-in-bio solution built with React.

## Features
- Custom profile and link management
- Ad monetization integration
- Analytics tracking
- Responsive design
- Self-hosted solution

```bash
npm install
npm run dev
```
